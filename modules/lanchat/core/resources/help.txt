List of commands:
/quit - exit LANChat
/save_config <host config> <peers config> - save the current config file
/show_peers - show current peers
/block <ip:port ...> - block certain peers
/unblock <ip:port...> - unblock certain peers
/help - display help screen
/encrypt <enable or disable> - enable/disable encryption
/whois <username> - show a peer's information
/chname <new username> - change username
/beep <enable or disable> - enables/disable beep when receiving messages
